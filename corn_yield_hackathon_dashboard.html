Cone of Uncertainty — Forecast Timeline
Aug 1
Early-Season Estimate — Widest Uncertainty
Inputs: NAIP July imagery · CDL corn mask · June–July GDD · precipitation anomaly vs 30-yr avg · NDVI from HLS. Analog year search: find years with similar June–July weather profile. Output: preliminary bu/acre range per state, wide confidence interval (±25%).
Sep 1
Mid-Season Update — Cone Narrows
Add August satellite imagery, silking/pollination stress signals, updated weather actuals. Recalculate analog year weights (penalize poor-fit years). Update regression model with new features. Confidence interval tightens to ±15%.
Oct 1
Late-Season — Near-Final Forecast
Dough/dent stage visible in imagery. Most weather uncertainty resolved. Analog years largely settled. Strong correlation with final NASS report. Confidence interval ±8%. State-level forecasts highly differentiated.
EOS
End-of-Season — Final Estimate vs NASS
Post-harvest NAIP if available, or final growing-season composite. Compare final model prediction to NASS published yield. Calculate RMSE and % error vs NASS. Cone collapses to near-point — benchmark for model accuracy.
Analog Year Algorithm (simplified)
def find_analog_years(target_weather, historical_df, n=5): """Find years most similar to current season weather profile.""" features = ['jun_gdd', 'jul_gdd', 'aug_gdd', 'jun_precip_anom', 'jul_precip_anom', 'aug_precip_anom'] X = historical_df[features].values target = target_weather[features].values.reshape(1, -1) # Cosine similarity across all historical years sims = cosine_similarity(target, X)[0] top_idx = np.argsort(sims)[::-1][:n] return historical_df.iloc[top_idx][['year', 'yield_bu_acre']].copy()

Prithvi Feature Extraction
from transformers import AutoModel import torch, rasterio import numpy as np def load_prithvi(device="cpu"): model = AutoModel.from_pretrained( "ibm-nasa-geospatial/Prithvi-100M", trust_remote_code=True ).to(device).eval() return model def extract_features(model, naip_tile_path, device="cpu"): """Extract spatial embeddings from a single NAIP tile.""" with rasterio.open(naip_tile_path) as src: # NAIP: bands 1=R, 2=G, 3=B, 4=NIR img = src.read().astype(np.float32) # (4, H, W) # Normalize per-band img = (img - img.mean(axis=(1,2), keepdims=True)) / \ (img.std(axis=(1,2), keepdims=True) + 1e-6) tensor = torch.tensor(img).unsqueeze(0).to(device) # (1,4,H,W) with torch.no_grad(): out = model(pixel_values=tensor) # Mean-pool patch embeddings → fixed-dim feature vector return out.last_hidden_state.mean(dim=1).squeeze().cpu().numpy()
